# Changelog

## 2.4.0

- Hardened backend auth, CORS, SSRF protection, tab/link ownership checks, and extension sync safety.
- Added Alembic migrations, Docker entrypoint migrations, CI, Makefile targets, and versioned extension packaging.
- Added revocable sessions, request IDs, security headers, readiness checks, and configurable rate limits.
- Added backup/restore modes, snapshots, tag management, duplicate merge with data preservation, and importers for JSON, browser bookmarks HTML, Pocket JSON, and Raindrop CSV.
- Added jobs, automatic snapshot support, full-text search index, public shared collections, recommendations, admin APIs, PWA service worker registration, and Playwright smoke tests.
