# LinkKeep Browser Extension

Save links to your LinkKeep instance with one click.

## Setup

1. Edit `config.js` — set your LinkKeep URL, username, password
2. Go to `chrome://extensions`
3. Enable "Developer mode"
4. Click "Load unpacked" → select this folder
5. Click the extension icon to save current page

## Features
- One-click save current tab
- Auto-fetches title, description, favicon
- Choose group (tab) before saving
- Login stored in chrome.storage
