# LinkKeep v2.0

Premium link manager with tabs, tags, favorites, search, and dark mode.

## Stack
- **Frontend:** React + Vite + Tailwind CSS + Framer Motion
- **Backend:** FastAPI + SQLAlchemy + SQLite
- **Infra:** Docker Compose + Nginx

## Features
- Tabbed link organization with colors
- Auto-fetch link metadata (title, description, favicon)
- Full-text search across all links
- Favorites with one-tap star
- Tags for cross-tab categorization
- Dark / light theme with system detection
- Mobile-first responsive design with bottom navigation
- PWA-ready

## Quick Start
```bash
docker compose up -d --build
```

Access at `http://localhost:9091`
